# 🚀 Client Order Fulfillment Guide

## 📁 Case A: If they bought the "Brand Kit & Pitch Deck" Gig (BrandForge Studio)

### Step 1: Collect Client Info
Ask the client for:
*   Their Company Name, Tagline, and Website.
*   Their target audience.
*   Their logo/brand colors (optional, since the tool can extract them).

### Step 2: Customize the Code (BrandForge)
1. Open the [app.js](file:///C:/Users/DAB/.gemini/antigravity-ide/brain/1d9040b7-b80f-4b0f-a498-f7250571d3d7/brand-forge-studio/app.js) file.
2. Edit the starting variables inside `brandData` to match the client's information:
   ```javascript
   let brandData = {
     name: "Client's Company Name",
     tagline: "Client's Tagline",
     mission: "Client's Mission statement...",
     archetype: "Creator", // Change to their archetype
     colors: {
       primary: "#8b5cf6", // Set their primary color
       secondary: "#ec4899",
       accent: "#10b981"
     }
     // ...
   };
   ```
3. Save the file.

### Step 3: Generate the Assets
1. Open [index.html](file:///C:/Users/DAB/.gemini/antigravity-ide/brain/1d9040b7-b80f-4b0f-a498-f7250571d3d7/brand-forge-studio/index.html) in your browser.
2. Navigate to **Visual Ad Canvas**, edit the headline text to represent their product, and click **Export Banner** to download their custom social media banners.
3. Go to **Pitch Deck Studio**, click through their 5-slide deck, press `Ctrl + P`, and save the slides as a professional presentation PDF.

### Step 4: Deliver on Fiverr
Deliver a `.ZIP` file containing:
*   Their customized **Brand Guidelines PDF**.
*   Their **5-Slide Presentation Pitch Deck PDF**.
*   The **Visual Banner Images** (PNGs) generated by the canvas.
*   *(Optional Premium Offer)*: Zip and send the modified `brand-forge-studio` source folder so they can host the tool themselves!

## 📁 Case B: If they bought the "Competitor Scraper" Gig (SpyGlass Portal)

### Step 1: Collect Competitor Details
Ask the client for:
*   Their own store product list and prices.
*   Their competitor store names and URLs.

### Step 2: Configure Their Inventory (SpyGlass)
1. Open the [app.js](file:///C:/Users/DAB/.gemini/antigravity-ide/brain/1d9040b7-b80f-4b0f-a498-f7250571d3d7/spyglass-intel/app.js) file.
2. Update the `myInventory` object with the client's actual product list and prices:
   ```javascript
   const myInventory = {
     electronics: [
       { id: 'e1', name: "Client's Real Product A", price: 499.00 },
       { id: 'e2', name: "Client's Real Product B", price: 129.00 }
     ]
     // Add their categories and item sets here
   };
   ```
3. Save the file.

### Step 3: Run and Export
1. Open [index.html](file:///C:/Users/DAB/.gemini/antigravity-ide/brain/1d9040b7-b80f-4b0f-a498-f7250571d3d7/spyglass-intel/index.html) in your browser.
2. Type in the competitor's store details and click **Trigger Live Scan**.
3. Let the tool calculate the price comparisons, then click **Export CSV** to download the clean data spreadsheet.
4. Save the visual report page as a PDF using `Ctrl + P` to give them a premium summary.

### Step 4: Deliver on Fiverr
Upload the completed files:
*   The **Competitor Price Matrix Spreadsheet** (`.csv` file).
*   The **Visual Price Audit Report PDF**.
*   *(Optional Premium Offer)*: Send them the complete, white-labeled `spyglass-intel` folder so they have their own private dashboard to run audits whenever they want!
